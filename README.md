BearField
=========
A small, efficient, easy to use [MongoDB][1] object layer built on top of
[PyMongo][2].

[![Build Status](https://travis-ci.org/WiFast/bearfield.svg?branch=master)](https://travis-ci.org/WiFast/bearfield)
